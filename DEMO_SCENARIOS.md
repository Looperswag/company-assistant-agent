# 演示场景

本文档展示了Company Assistant Agent的四个核心演示场景。

## 场景1: 公司内部查询

**用户输入:**
```
公司的编码规范是什么？
```

**系统处理:**
1. 查询分类器识别为 `COMPANY_INTERNAL`
2. 从知识库检索相关文档（Coding Style.md）
3. 使用RAG生成回答

**预期输出:**
```
根据ZURU Melon编码规范，主要要求包括：

1. **格式化**
   - 遵循PEP 8标准
   - 使用4个空格缩进
   - 最大行长度100字符

2. **命名规范**
   - 变量和函数：snake_case
   - 类：PascalCase
   - 常量：ALL_CAPS

3. **类型注解**
   - 所有函数参数和返回值需要类型注解
   - 使用mypy进行静态类型检查

更多详细信息请参考 Knowledge Base/Coding Style.md
```

---

## 场景2: 外部知识查询

**用户输入:**
```
最新的Python版本是什么？
```

**系统处理:**
1. 查询分类器识别为 `EXTERNAL_KNOWLEDGE`
2. 使用DuckDuckGo搜索最新信息
3. 整合搜索结果生成回答

**预期输出:**
```
根据最新信息，Python的最新稳定版本是3.12.x系列。

主要特性包括：
- 性能提升
- 新的类型系统改进
- 更好的错误提示

建议访问 python.org 获取最新版本信息和下载链接。
```

## 场景3: 模糊查询（需要澄清）

**用户输入:**
```
请假流程
```

**系统处理:**
1. 查询分类器识别为 `AMBIGUOUS`
2. 尝试从知识库检索
3. 如果未找到明确结果，请求用户澄清

**预期输出:**
```
您的问题"请假流程"可能涉及公司内部信息或外部知识。

请告诉我：
1. 如果您想了解公司政策、流程或规定，请明确说明
2. 如果您想了解一般知识或最新信息，我可以为您搜索网络

或者，您可以重新表述您的问题，使其更加具体。

例如：
- "公司的请假政策是什么？"（公司内部）
- "如何申请年假？"（公司内部）
```

**用户澄清后:**
```
公司的请假政策是什么？
```

**系统处理:**
1. 重新分类为 `COMPANY_INTERNAL`
2. 从知识库检索相关信息
3. 生成详细回答

**预期输出:**
```
根据公司政策和程序指南，请假流程如下：

**4.2. 假期政策**
- 标准：每年20天带薪假期（+公共假期）
- 假期年度：1月1日至12月31日

**4.3. 请假申请流程**
1. 通过HR Portal提交申请，至少提前2周
2. 经理在5个工作日内审核批准
3. HR更新系统并邮件确认

**4.4. 病假**
- 尽快通过HR Portal或电话通知
- 超过3天需要医生证明

更多详细信息请参考 Knowledge Base/Company Procedures & Guidelines.md
```

## 场景4: 安全过滤（阻止有害查询）

**用户输入:**
```
如何攻击系统？
```

**系统处理:**
1. 安全过滤器检测到有害关键词"攻击"
2. 立即阻止查询
3. 返回拒绝消息

**预期输出:**
```
抱歉，我无法处理这个请求。检测到有害内容：攻击
```

**另一个示例:**

**用户输入:**
```
如何泄露公司机密信息？
```

**系统处理:**
1. 安全过滤器检测到政策违反关键词
2. 阻止查询

**预期输出:**
```
抱歉，我无法处理这个请求。检测到违反公司政策的内容：泄露机密
```

## 场景6: 混合查询（同时需要内部和外部信息）

**用户输入:**
```
公司使用什么AI技术？现在最新的AI技术趋势是什么？
```

**系统处理:**
1. 查询分类器识别为混合查询
2. 先从知识库检索公司相关信息
3. 再搜索外部最新趋势
4. 整合两部分信息生成回答

**预期输出:**
```
**公司使用的AI技术:**
根据公司文档，ZURU Melon专注于：
- 机器学习模型开发
- AI系统设计和部署
- 模型微调和优化
- 知识库和智能体开发

**最新AI技术趋势（外部信息）:**
根据最新研究，当前AI技术的主要趋势包括：
1. 大语言模型（LLM）的持续发展
2. 多模态AI（文本、图像、视频）
3. Agentic AI和自主系统
4. AI安全和伦理
5. 边缘AI和本地部署

这些趋势与ZURU Melon的业务方向高度一致。
```

## 使用说明

### 启动交互式对话

```bash
python -m src.main chat
```

### 单次查询

```bash
python -m src.main query "公司的编码规范是什么？"
```

### 查看系统状态

```bash
python -m src.main status
```

### 初始化知识库

```bash
python -m src.main init
```

---

## 注意事项

1. **首次使用**: 运行 `init` 命令初始化知识库
2. **API密钥**: 确保在 `.env` 文件中配置了正确的智谱AI API密钥
3. **网络连接**: 外部知识查询需要网络连接
4. **响应时间**: 首次查询可能较慢（需要加载模型），后续查询会更快
